- Jam3의 Medal of Honor: Above & Beyond (https://www.oculus.com/medalofhonor)

## 5. 씬 구성 상세

상세 내용은 개발자와 함께 조정하며 진행한다.

### Scene 1 — 상공 (scroll 0% ~ 25%)

**카메라 위치**: `(0, 200, 0)`, 정면으로 광주 시내를 내려다봄
**렌더링**:

- OSM 건물 데이터를 `ExtrudeGeometry`로 돌출시킨 추상 도시 격자
- 건물 높이는 OSM `building:levels` 태그 활용, 없으면 랜덤 1~4 (seed 고정)
- 전체 색상: `#1a1a1a` 모노톤. 도로만 `#2a2a2a`로 구분

**셰이더**: grain 강도 `0.12`, 흑백 100%
**텍스트 등장**:

```
"1980년 5월 18일"  — GSAP SplitText, 글자 단위 stagger 0.05s
"광주"             — 0.3s 후 fade in
```

### Scene 2 — 하강 전환 (scroll 25% ~ 55%)

**카메라 경로**: `(0, 200, 0)` → `(0, 8, 60)`, CatmullRomCurve3 보간
**카메라 pitch**: 정면 내려봄(90°) → 수평(0°)으로 회전
**파티클**: 500개, 카메라 주변 원통 분포, Y축 위로 스쳐 지나가는 효과 (상대 속도감)
**셰이더 전환**: `uProgress` 0.25→0.55 구간에서 grain 0.12→0.08, 흑백→세피아(midpoint)
**텍스트**: 없음 (전환 집중)

### Scene 3 — 거리 (scroll 55% ~ 80%)

**카메라 위치**: `(0, 2, 80)` ~ `(0, 2, 110)`, 금남로 방향 직진
**1인칭 인터랙션**: `mousemove` 이벤트로 카메라 yaw ±8°, pitch ±4° (GSAP quickTo)
**타임라인 텍스트**: `timeline.json`의 각 항목이 스크롤 진행에 따라 순서대로 등장

- 날짜: 상단 좌측, 모노스페이스
- 증언: 화면 하단, NotoSerifKR, 최대 2줄
- 등장: GSAP `fromTo`, opacity 0→1, y 20→0, duration 0.6

**셰이더**: grain 0.06, 비네팅 강도 0.4

### Scene 4 — 귀결 (scroll 80% ~ 100%)

**카메라**: 천천히 상승 `(0, 2, 110)` → `(0, 60, 110)`, 뒤로 빠지며 도시 전체 보임
**셰이더**: 흑백 → 컬러 완전 전환 (`smoothstep(0.8, 1.0, uProgress)`)
**도시 색상**: 건물이 `#1a1a1a` → `#8B4513` (세피아 브라운) 으로 서서히 변환
**최종 텍스트**:

```
"우리가 기억합니다"  — 중앙, 2.0s fade in
"5·18민주화운동"    — 하단, 0.8s delay
```

### 6-2. 씬 루트 + 카메라 릭 (Experience.jsx / CameraRig.jsx)

```jsx
// Experience.jsx
import { EffectComposer } from '@react-three/postprocessing';
import CameraRig from './CameraRig';
import CityMesh from './CityMesh';
import Particles from './Particles';
import { FilmEffect } from '../shaders/FilmEffect';

export default function Experience() {
  return (
    <>

      <Particles />
      <ambientLight intensity={0.4} />
      <EffectComposer>
        <FilmEffect />
      </EffectComposer>
    </>
  );
}

// CameraRig.jsx
import { useFrame, useThree } from '@react-three/fiber';
import { useScroll } from '@react-three/drei';
import { cameraPath } from '../utils/cameraPath';

export default function CameraRig() {
  const { camera } = useThree();
  const scroll = useScroll(); // scroll.offset: 0 ~ 1

  useFrame(() => {
    const t = scroll.offset;
    const pos = cameraPath.getPoint(t);
    const look = cameraPath.getPoint(Math.min(t + 0.01, 1));
    camera.position.copy(pos);
    camera.lookAt(look);
  });

  return null;
}
```

### 6-3. 필름 셰이더 Effect 래퍼 (FilmEffect.jsx)

```jsx
import { useRef } from "react";
import { useScroll } from "@react-three/drei";
import { useFrame } from "@react-three/fiber";
import { Effect } from "postprocessing";
import { Uniform } from "three";
import fragmentShader from "./film.frag.glsl?raw";

class FilmEffectImpl extends Effect {
  constructor() {
    super("FilmEffect", fragmentShader, {
      uniforms: new Map([
        ["uProgress", new Uniform(0)],
        ["uTime", new Uniform(0)],
      ]),
    });
  }
}

export function FilmEffect() {
  const effectRef = useRef(new FilmEffectImpl());
  const scroll = useScroll();

  useFrame(({ clock }) => {
    effectRef.current.uniforms.get("uProgress").value = scroll.offset;
    effectRef.current.uniforms.get("uTime").value = clock.elapsedTime;
  });

  return <primitive object={effectRef.current} />;
}
```

### 6-4. 필름 셰이더 GLSL (film.frag.glsl)

```glsl
uniform float uProgress;
uniform float uTime;

float random(vec2 st) {
  return fract(sin(dot(st.xy, vec2(12.9898, 78.233))) * 43758.5453123);
}

// @react-three/postprocessing Effect는 mainImage 시그니처를 사용한다
// void main() + texture2D(uScene) 방식과 다르므로 주의
void mainImage(const in vec4 inputColor, const in vec2 uv, out vec4 outputColor) {
  vec3 color = inputColor.rgb;

  float gray = dot(color, vec3(0.299, 0.587, 0.114));
  vec3 bw = vec3(gray);

  vec3 sepia = vec3(
    dot(color, vec3(0.393, 0.769, 0.189)),
    dot(color, vec3(0.349, 0.686, 0.168)),
    dot(color, vec3(0.272, 0.534, 0.131))
  );

  // 0.0~0.5: 흑백, 0.5~0.75: 세피아, 0.75~1.0: 컬러
  float colorMix = smoothstep(0.75, 1.0, uProgress);
  float sepiaMix = smoothstep(0.5, 0.75, uProgress) * (1.0 - colorMix);
  vec3 blended = mix(bw, sepia, sepiaMix);
  blended = mix(blended, color, colorMix);

  float grainIntensity = mix(0.12, 0.04, uProgress);
  float grain = random(uv + fract(uTime * 0.1)) - 0.5;
  blended += grain * grainIntensity;

  float vignetteStrength = mix(0.5, 0.2, uProgress);
  vec2 center = uv - 0.5;
  float vignette = 1.0 - dot(center, center) * vignetteStrength * 4.0;
  blended *= vignette;

  outputColor = vec4(blended, inputColor.a);
}
```

### 6-6. 텍스트 오버레이 (ui/TextOverlay.jsx) — GSAP 담당

```jsx
import { useEffect, useRef } from "react";
import gsap from "gsap";
import { SplitText } from "gsap/SplitText";

gsap.registerPlugin(SplitText);

export default function TextOverlay() {
  const titleRef = useRef();

  useEffect(() => {
    const split = new SplitText(titleRef.current, { type: "chars" });
    gsap.from(split.chars, {
      opacity: 0,
      y: 20,
      stagger: 0.05,
      duration: 0.8,
      ease: "power2.out",
    });
  }, []);

  return (
    <div
      style={{
        position: "fixed",
        inset: 0,
        pointerEvents: "none",
        zIndex: 10,
        fontFamily: "'Noto Serif KR', serif",
        color: "#e8e0d0",
      }}
    >
      <div
        ref={titleRef}
        style={{
          position: "absolute",
          top: "50%",
          left: "50%",
          transform: "translate(-50%, -50%)",
          fontSize: "2rem",
          textAlign: "center",
        }}
      >
        1980년 5월 18일
      </div>
    </div>
  );
}
```

### 6-7. OSM 데이터 로더 (utils/osmLoader.js)

```javascript
import osmtogeojson from "osmtogeojson";

const BBOX = "35.140,126.900,35.155,126.930";
const QUERY = `
[out:json][timeout:25];
(
  way["building"](${BBOX});
  way["highway"](${BBOX});
);
out body;
>;
out skel qt;
`;

export async function loadGwangjuOSM() {
  const url = `https://overpass-api.de/api/interpreter?data=${encodeURIComponent(QUERY)}`;
  const res = await fetch(url);
  const osmData = await res.json();
  return osmtogeojson(osmData);
}
```

### 6-8. GeoJSON → Three.js 좌표 변환 (utils/geoToThree.js)

```javascript
const ORIGIN_LAT = 35.1468;
const ORIGIN_LNG = 126.9172;
const SCALE = 5000;

export function lngLatToXZ(lng, lat) {
  const x = (lng - ORIGIN_LNG) * SCALE * Math.cos((ORIGIN_LAT * Math.PI) / 180);
  const z = -(lat - ORIGIN_LAT) * SCALE;
  return { x, z };
}
```
