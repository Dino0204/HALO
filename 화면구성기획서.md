### HUD-A: 우측 타임라인 (날짜 진행 표시)

**위치**: 화면 우측 고정, 수직 중앙 정렬
**표시 방식**: 시작일 · 현재일(강조) · 종료일을 수직 선으로 연결

```
5.18        ← 항쟁 시작일, 항상 표시, 작은 크기, opacity 0.5
 ㅣ
 ㅣ         ← 진행 선: 현재 씬 위치에 따라 채워짐 (흰색 → 회색)
 ㅣ
5.21        ← 현재 씬의 날짜, 크고 밝게 강조 (2× 크기, 흰색)
 ㅣ
 ㅣ
 ㅣ
5.27        ← 항쟁 종료일, 항상 표시, 작은 크기, opacity 0.5
```

**씬별 날짜 데이터**

| --- | ------------------- | ---------------------------- |

| 01  | 1979.12.12          | — (항쟁 전, 별도 스타일)     |
| 02  | 1980년 봄           | —                            |
| 03  | 5.18                | 강조                         |
| 04  | 5.18                | 강조                         |
| 05  | 5.20                | 강조                         |
| 06  | 5.21                | 강조                         |
| 07  | 5.21                | 강조                         |
| 08  | 5.27                | 강조 (종료일과 동일, 붉은색) |
| 09  | 5.27                | 강조                         |

> 별도의 "배경" 레이블로 대체한다.
> 씬 03부터 타임라인이 fade in되어 등장한다.

**React 컴포넌트 구조**

```jsx
// ui/HUDTimeline.jsx
export default function HUDTimeline({ currentScene }) {
  const sceneData = SCENE_HUD_DATA[currentScene];
  if (!sceneData || !sceneData.visible) return null;

  const progress = DATE_PROGRESS[sceneData.date] ?? 0;
  const isFinal = sceneData.date === "5.27";

  return (
    <div className="hud-timeline">
      {/* 시작일 */}
      <span className="hud-date-edge">5.18</span>

      {/* 위쪽 선 — 시작일 ~ 현재 */}
      <div className="hud-line-track" style={{ height: "100px" }}>
        <div
          className="hud-line-fill"
          style={{ height: `${progress * 100}%` }}
        />
      </div>

      {/* 현재 날짜 강조 */}
      <span className={`hud-date-current ${isFinal ? "final" : ""}`}>
        {sceneData.date}
      </span>

      {/* 아래쪽 선 — 현재 ~ 종료일 */}
      <div className="hud-line-track" style={{ height: "100px" }}>
        <div
          className="hud-line-fill"
          style={{ height: `${(1 - progress) * 100}%`, top: "auto", bottom: 0 }}
        />
      </div>

      {/* 종료일 */}
      <span className="hud-date-edge">5.27</span>
    </div>
  );
}
```

### TextOverlay.jsx 통합 구조 (HUD 포함)

```jsx
// ui/TextOverlay.jsx
import HUDTimeline from "./HUDTimeline";
import HUDLocation from "./HUDLocation";
import { useCurrentScene } from "../scene/Experience";

export default function TextOverlay() {
  const currentScene = useCurrentScene();

  return (
    <div
      style={{ position: "fixed", inset: 0, pointerEvents: "none", zIndex: 10 }}
    >
      {/* HUD-A: 우측 타임라인 */}
      <HUDTimeline currentScene={currentScene} />

      {/* HUD-B: 좌측 상단 위치 */}
      <HUDLocation currentScene={currentScene} />

      {/* 씬별 중앙 텍스트 — 별도 컴포넌트에서 관리 */}
      <SceneText currentScene={currentScene} />
    </div>
  );
}
```

> `useCurrentScene`은 `Experience.jsx`에서 export하되,
> Three.js 컨텍스트 외부(`TextOverlay`)에서 사용하려면
> `zustand` 또는 React `Context`로 씬 상태를 공유해야 한다.
> 가장 단순한 구현: `zustand`의 `useSceneStore`를 만들어
> `CameraRig.jsx`의 `useFrame` 안에서 씬 번호를 set하고
> `TextOverlay.jsx`에서 구독한다.

```javascript
// store/sceneStore.js
import { create } from "zustand";

export const useSceneStore = create((set) => ({
  currentScene: 0,
  setScene: (scene) => set({ currentScene: scene }),
}));
```

## Scene 01 — 12·12 군사 쿠데타: 서울 점령

**scroll 9% ~ 18%**

### 시각 요소

- **지도 위 애니메이션**
  - 서울 위치에 빨간 점(`THREE.SphereGeometry`, radius 0.3) 등장
  - 서울을 중심으로 동심원 파문이 퍼져나감
    - `THREE.RingGeometry`를 `useFrame`에서 scale 1→4, opacity 1→0으로 반복
    - 파문 3겹, 0.3s stagger
  - 파문과 함께 서울 점 색상이 `#ff3333`으로 강해짐

- **지도 전체**
  - 서울을 제외한 나머지 지도 색상이 `#1a1a1a`로 더 어두워짐
  - 어둠이 서울에서 바깥쪽으로 퍼지는 효과 (거리 기반 uniform)

- **셰이더**: 흑백 유지, grain 0.12 (긴장감)

### 텍스트 오버레이

```
[좌측 상단, 날짜 태그]
"1979.12.12"
— monospace, #ff6666, 즉시 등장

[중앙]
"전두환, 군사 반란을 일으키다"
— SplitText, 단어 단위 stagger

[중앙 하단]
"군부가 서울을 장악했다.
 국회는 봉쇄되고, 민주주의는 멈췄다."
— Noto Serif KR, 2줄, fade in 0.6s delay
```

### Three.js 구현 힌트

```jsx
// 서울 좌표 → Three.js 로컬 좌표 (한국 지도 기준)
// 지도 중심을 원점(0,0)으로 정규화한 뒤 적용
const SEOUL_POS = new THREE.Vector3(-0.8, 2.1, 0); // 지도 메시 좌표계 기준

// 파문 링
function PulseRing({ position, delay }) {
  const ref = useRef();
  useFrame(({ clock }) => {
    const t = (clock.elapsedTime - delay) % 2.0;
    ref.current.scale.setScalar(1 + t * 1.5);
    ref.current.material.opacity = Math.max(0, 1 - t / 2.0);
  });
  return (
    <mesh ref={ref} position={position} rotation={[-Math.PI / 2, 0, 0]}>
      <ringGeometry args={[0.3, 0.35, 32]} />
      <meshBasicMaterial color="#ff3333" transparent opacity={1} />
    </mesh>
  );
}
```

## Scene 03 — 한국 지도 → 광주 확대 → 전남대 확대

**scroll 27% ~ 36%**

### 시각 요소 — 3단계 카메라 줌

**3-1. 지도에서 광주로 줌인**

- 카메라가 광주 마커를 향해 부드럽게 이동

### 최루탄 파티클 구현 힌트

```jsx
// Particles.jsx
const PARTICLE_COUNT = 800;

export default function TearGasParticles({ active }) {
  const points = useRef();
  const positions = useMemo(() => {
    const arr = new Float32Array(PARTICLE_COUNT * 3);
    for (let i = 0; i < PARTICLE_COUNT; i++) {
      arr[i * 3] = (Math.random() - 0.5) * 20; // x
      arr[i * 3 + 1] = Math.random() * 5; // y
      arr[i * 3 + 2] = (Math.random() - 0.5) * 10; // z
    }
    return arr;
  }, []);

  useFrame(({ clock }) => {
    const t = clock.elapsedTime;
    const pos = points.current.geometry.attributes.position.array;
    for (let i = 0; i < PARTICLE_COUNT; i++) {
      pos[i * 3 + 1] += 0.005; // 상승
      if (pos[i * 3 + 1] > 12) pos[i * 3 + 1] = 0; // 리셋
    }
    points.current.geometry.attributes.position.needsUpdate = true;
  });

  return (
    <points ref={points}>
      <bufferGeometry>
        <bufferAttribute
          attach="attributes-position"
          array={positions}
          count={PARTICLE_COUNT}
          itemSize={3}
        />
      </bufferGeometry>
      <pointsMaterial
        size={0.08}
        color="#cccccc"
        transparent
        opacity={0.6}
        sizeAttenuation
        depthWrite={false}
      />
    </points>
  );
}
```

```jsx
  [0.54, 0.63], // Scene 06
  [0.63, 0.72], // Scene 07
  [0.72, 0.81], // Scene 08
  [0.81, 0.9], // Scene 09
  [0.9, 1.0], // Scene 10
];

export function useCurrentScene() {
  const scroll = useScroll();
  const [currentScene, setCurrentScene] = useState(0);

  useFrame(() => {
    const t = scroll.offset;
    const idx = SCENE_RANGES.findIndex(([start, end]) => t >= start && t < end);
    if (idx !== -1 && idx !== currentScene) setCurrentScene(idx);
  });

  return currentScene;
}

// 씬 내 로컬 진행도 (0~1)
export function useSceneProgress(sceneIndex) {
  const scroll = useScroll();
  const [start, end] = SCENE_RANGES[sceneIndex];
  return Math.max(0, Math.min(1, (scroll.offset - start) / (end - start)));
}
```

Three.js에서 GeoJSON → ShapeGeometry 변환:

```javascript
import { Feature } from "geojson";

function geoJsonToShape(feature) {
  const shape = new THREE.Shape();
  const coords = feature.geometry.coordinates[0];
  coords.forEach(([lng, lat], i) => {
    const x = (lng - 127.5) * 2.5;
    const y = (lat - 36.5) * 3.0;
    i === 0 ? shape.moveTo(x, y) : shape.lineTo(x, y);
  });
  return shape;
}
```
