# 5·18민주화운동 인터랙티브 웹 경험

스크롤로 1980년 5월 광주의 14개 장면을 순서대로 체험하는 Three.js + React 웹 경험.  
항공 시점에서 시작해 거리로 강하하며 역사적 사건에 몰입하는 스크롤-카메라 연동 방식.

## 주요 기능

- **스크롤 카메라**: `ScrollControls` + `CatmullRomCurve3` 경로로 항공 시점 → 1인칭 시점 전환
- **필름 셰이더**: 스크롤 진행에 따라 흑백 → 세피아 → 컬러 전환 + 필름 그레인 + 비네팅
- **HUD 타임라인**: 우측 고정 날짜 진행 표시 (5.18 → 5.27)
- **자동재생 / 배속**: 자동으로 씬을 진행하며 재생 속도 조절 가능
- **처음 / 끝 버튼**: 첫 씬 또는 마지막 씬으로 즉시 이동
- **로딩 화면**: GLB 모델 및 OSM 데이터 로드 중 진행 표시

## 개발

```bash
bun install
bun run dev      # http://localhost:5173
bun run build
```
