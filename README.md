# 5·18민주화운동 인터랙티브 웹 경험

스크롤에 따라 1980년 5월 광주의 14개 장면을 차례로 보여주는 Three.js + React 웹 경험.

## 개발

```bash
bun install
bun run dev      # http://localhost:5173
bun run build
```
